# 🌍 Language Translation Tool

A Python based desktop application that translates text between multiple languages using Google Translate integration. The application includes text-to-speech functionality, language swapping, clipboard support, and a modern user-friendly interface with Dark Mode support.

## ✨ Features

* 🌐 Translate Text Between Multiple Languages
* 🔄 Source and Target Language Selection
* 🎤 Text-to-Speech (TTS) Support
* 📋 Copy Translated Text to Clipboard
* 🔁 Swap Languages Instantly
* 🌙 Dark Mode Support
* 🖥️ Interactive Tkinter GUI
* 📚 Dynamic Language Loading
* 🔍 Supports Hundreds of Languages
* ⚡ Fast Translation Processing
* 📱 Responsive and Resizable Interface

## 🛠️ Technologies Used

* Python
* Tkinter
* Deep Translator
* Google Translate API
* gTTS (Google Text-to-Speech)
* Playsound
* Pyperclip
* Multithreading

## 🚀 How to Run

1. Clone or download the repository.
2. Install the required dependencies:

```bash
pip install deep-translator gtts playsound pyperclip
```

3. Run the application:

```bash
python language_translation_tool.py
```

4. Select source and target languages.
5. Enter text and click Translate.
6. Copy or listen to the translated text.
